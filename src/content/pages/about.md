## 👋 你好，Brandon

我是一位熱愛透過代碼解決生活問題的**軟體工程師**。

我深信技術不應該只存在於產線或機房，更應該走入日常生活。對我來說，開發Side Projects不只是練習技術的沙盒，更是我將靈感具現化的過程。

## 🛠️ 目前正在進行的計畫

### 👕 衣起記 (Digital Wardrobe)

一個專為穿搭愛好者設計的數位衣櫥管理工具，旨在幫助使用者數位化自己的穿著選擇，實踐更有計畫的個人管理。

## 💻 技術棧 (Tech Stack)

我喜歡保持學習的慣性，目前我主要使用的工具包括：

- **Frontend**: React (Next.js), Astro, Tailwind CSS, React Native
- **Backend**: Supabase (PostgreSQL, Auth, Edge Functions)
- **Language**: TypeScript, C#, Python
- **Others**: Git, Vercel 部署自動化

## ✍️ 為什麼我寫這個部落格？

在開發過程中，我經常遇到一些「幽靈級」的 Bug（例如 Supabase 的 API 權限陷阱）。我發現將這些踩坑紀錄寫下來，不僅能幫助三個月後的自己，也能為同樣卡關的開發者提供一點方向。
